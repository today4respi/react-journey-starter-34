JenCity API Documentation

Documentation complète des API pour l'application JenCity

Configuration du Serveur

http://localhost:3000/api

Users

GET
/api/users
Récupère la liste de tous les utilisateurs avec leurs informations de base comme le nom, l'email et le rôle.

Réponse

Status: 200
JSON 
{
  "status": 200,
  "data": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com",
      "role": "user"
    },
    {
      "id": 2,
      "name": "Jane Smith",
      "email": "jane@example.com",
      "role": "admin"
    }
  ]
}
GET
/api/users/:id
Récupère les informations détaillées d'un utilisateur en fonction de son identifiant unique.

Paramètres

Nom Type Obligatoire Description
id number Oui L'identifiant unique de l'utilisateur

Réponse

Status: 200
JSON 
{
  "status": 200,
  "data": {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com",
    "role": "user",
    "createdAt": "2023-06-15T10:30:00Z"
  }
}
POST
/api/users
Crée un nouvel utilisateur dans la base de données avec un rôle par défaut "user".

Corps de la Requête

JSON 
{
  "name": "New User",
  "email": "newuser@example.com",
  "password": "securepassword123"
}

Réponse

Status: 201
JSON 
{
  "status": 201,
  "data": {
    "id": 3,
    "name": "New User",
    "email": "newuser@example.com",
    "role": "user",
    "createdAt": "2023-08-20T15:45:00Z"
  }
}
PUT
/api/users/:id
Modifie les informations d'un utilisateur existant comme le nom et l'email.

Paramètres

Nom Type Obligatoire Description
id number Oui L'identifiant unique de l'utilisateur

Corps de la Requête

JSON 
{
  "name": "Updated Name",
  "email": "updated@example.com"
}

Réponse

Status: 200
JSON 
{
  "status": 200,
  "data": {
    "id": 1,
    "name": "Updated Name",
    "email": "updated@example.com",
    "role": "user",
    "updatedAt": "2023-08-21T09:12:00Z"
  }
}
DELETE
/api/users/:id
Supprime définitivement un utilisateur de la base de données.

Paramètres

Nom Type Obligatoire Description
id number Oui L'identifiant unique de l'utilisateur

Réponse

Status: 204
JSON 
{
  "status": 204,
  "message": "User deleted successfully"
}
